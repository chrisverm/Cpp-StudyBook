Overloading and Overriding

Function overloading

Overloading a function is the ability for multiple functions of the same name in the same scope to be declared and defined as long as these functions have different parameters. Overloading requires functions to have different method signatures and, as shown below, can occur without inheritance :

#include <iostream> void Print( int i ) { std::cout << i; } void Print( char* str ) { std::cout << str; } int main( void ) { Print( 17 ); // outputs "17" Print( "Hellow World!" ); // outputs "Hello World!" return 0; }

Function overriding

Overriding a function, on the other hand, requires two functions to have the same signature (that is, the same function name and the same parameters). Overriding requires inheritance and is used to give different meaning an inherited function. For example :

#include <iostream> class Animal { protected: virtual void Move( int meters ); }; class Bird : public Animal { public: void Move( int meters ) { std::cout << "Flying " << meters << " meters"; } }; class Dog : public Animal { public: void Move( int meters ) { std::cout << "Running " << meters << " meters"; } };
